## Code restructuring

### `packages/daemon/src/server.ts` (7,597 lines)

**Charter & current responsibilities.** The monolithic Bun HTTP server: every API route (70+), all module-level state (caches, timers, SSE registry, single-flight guards), the WebSocket terminal I/O handler, startup/shutdown lifecycle, and background sampling loops. Pure helpers, a favicon proxy subsystem, the `/api/repos` NDJSON cache+streamer, and PTY grace-timer logic are all embedded in module scope.

**Design decisions to preserve** (sorted by riskIfLost):

*High risk:*
- **TDZ ordering** — `shellCwds`, `shellTermIds`, `graceTimers` and their consts MUST stay declared above `Bun.serve()` (1498–1517). Route handlers close over them; a request landing before a later `const` throws `ReferenceError`. Pinned by `server-resilience.test.ts`.
- **CORS allowlist is an explicit `Set`, never `*`** (601–720). The security boundary against localhost exfiltration. `SUPERGIT_EXTRA_ORIGINS` is the only sanctioned extension.
- **`peerModeEnabled=false` → 403 for non-loopback before any route** (1619–1624). Primary LAN gate.
- **`repsCacheGen` generation counter** (831, 1119–1201) — a slow in-flight fan-out must not overwrite a cache invalidated mid-flight. Increment must stay atomic with `invalidateReposCache()`.
- **`/api/session` stat()-based ETag fast-path before `getSessionResponseJson`** (2544–2578). Polled ~22Hz; load-bearing.
- **`idleTimeout = 30s`** (1561–1568) — guards a Bun 1.3.x listener-wedge regression. Pinned by `server-resilience.test.ts`.
- **zsh `procName` must contain "zsh"** else prepend `zsh-` (2790–2803) — preserves zsh emulation mode. Pinned by `terminals.test.ts:415`.
- **`resolveSessionAgent` is the single `?source=` allowlist gate** (383–420) — without it, session routes become arbitrary file-read endpoints. The imported-sessions `length-2` segment extraction is critical.

*Medium risk:* abort-but-finish behavior in `reposNDJSONFresh` (1003–1163); `/api/active-sends` revision ETag (4978–4991); `maxRequestBodySize = 50MB` (1555); PTY grace timer = 60s (1489); `ollamaChatAborts` per-termId cancel (3195); `repoSummaryInflight` single-flight (5775); `shutdown()` re-entry guard + 2s hard-exit (7412–7436); `SUPERGIT_NO_UI_DIR` guard (185–196).

*Low risk:* `sampleShell*` early-return on `sseSubscribers.size===0` (7529); `notifyOfferStatus` deliberate no-op (4062); `/api/stream` cancel empty-try (6378); bun.lock cleanup (2677).

- **Shell `__new__:shell:`→`__attached__:shell:<termId>` swap on `on:spawn`** (8589–8613) — prevents duplicate PTY after reload. Claude/Codex use `resumeSessionId` instead.

- **SSE `change` gated on `changeKindRequiresReposReload/EventsReload`** (4428–4436) — skips 500–1000ms rebuilds for chatty events.
